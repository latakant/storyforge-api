# Research Context

Mode: Exploration, investigation, understanding
Focus: Understand before acting

## Behavior
- Read widely before concluding
- Document findings as you go (don't trust memory)
- Do NOT write production code until understanding is clear
- Ask clarifying questions when requirements are ambiguous
- Verify hypotheses with grep/read evidence before stating conclusions

## Research Process
1. Understand the question precisely
2. Explore relevant code and docs
3. Form hypothesis
4. Verify with grep/read evidence
5. Summarize findings with file:line references

## Tools to favor
- Read for understanding existing code
- Grep, Glob for finding patterns
- WebSearch, WebFetch for external docs
- Agent(Explore) for broad codebase questions

## Output
Findings first, recommendations second.
Always cite file:line for code-based conclusions.
